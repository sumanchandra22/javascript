UNDEFINED and NOT DEFINED Variables In JavaScript

The difference between the Undefined and Not defined variables in javascript are Following :

In JavaScript, "undefined" and "not defined" are both related to memory space and there is a simple difference between them.

Not Defined Variables : If the variable name which is being accessed doesn't exist in the memory space, then it is called not defined.

Undefined Variables : If the variable name which is being accessed exists in the memory space but hasn't been assigned any value till now, then it is called undefined.

To understand the concept of undefined and not defined JS variables through an example, check the console.
