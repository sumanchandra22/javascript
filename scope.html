Scope Of Varibles In JavaScript


JavaScript variables have only two scopes - Global Scope and Local Scope.

Global Scope : A global variable has global scope which means it can be defined anywhere in the JavaScript code.

Local Scope : A local variable will be visible only within a function where it is defined. Function parameters are always local to that function.

To understand the concept of scope of JS variables through an example, check the console.
